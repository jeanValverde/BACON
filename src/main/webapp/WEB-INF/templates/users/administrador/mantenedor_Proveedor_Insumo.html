¿Cómo se relacionan los Proveedores con los insumos? (Insumos del Proveedor)

    Para agregar un insumo al Proveedor debes:

  1. Presiona busca el insumo que desees en la lista.

  2. Ingresa un Precio dentro del Rango (1000 a 5000000)
  3. Presiona el botón "Agregar" para finalizar.

    Para editar un insumo relacionado con el proveedor debes:

  1. Presiona y buscar el insumo a editar.

  2. Cambiar el valor en la entrada del precio (Ejemplo: 5000)
  3. Presiona el botón "Editar"

    Para eliminar un insumo relacionado con el proveedor debes:

  1. Presiona y busca el insumo a eliminar.

  2. Presiona el botón "Eliminar" (Al eliminar un insumo relacionado con el proveedor los datos serán irrecuperables, pero el insumo no verá afectado)
